# wiki帮助中心 模板

## 介绍

通过知识库同步页面内容，从而实现内容和站点分离；用于大规模可重用文档内容管理，如Wiki、帮助文档和内容门户。

demo: https://demo-wiki.uibak.com/

教程： https://help.baklib.cn/themes/wiki

## 开发

如何通过 git 将模板挂载到 Baklib: https://dev.baklib.cn/guide/git
